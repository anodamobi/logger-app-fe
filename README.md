## About The Project

This is anoda logger created to save logs from your apps to database and send
it back for you on request in user-friendly manner.

### Built With

- [React.js](https://reactjs.org/)
- [Mantine](https://mantine.dev/)

<!-- GETTING STARTED -->

### Prerequisites

- Node ^16
- Mantine ^4

### Local installation

`npm install`

`npm run start`

### Production run

`npm install`

`npm run build`
